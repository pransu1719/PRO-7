# Multi-Utility Toolkit 🛠️

A simple Python console-based application that bundles together everyday utility operations — datetime tools, math operations, random data generation, UUID generation, and file handling — all in one menu-driven program.

## 📋 Features

- **Datetime and Time Operations**
  - Display current date and time
  - Calculate difference between two dates
  - Format date into a custom format
  - Stopwatch
  - Countdown timer

- **Mathematical Operations**
  - Calculate factorial
  - Solve compound interest
  - Trigonometric calculations (sin, cos, tan)
  - Area of a circle

- **Random Data Generation**
  - Generate random number
  - Generate random list
  - Create random password
  - Generate random OTP

- **Generate UUID**
  - Generate a unique UUID4 identifier

- **File Operations**
  - Create a new file
  - Write to a file
  - Read from a file
  - Append to a file

- **Explore Module Attributes**
  - View available attributes of `math`, `random`, and `datetime` modules

## 🧰 Requirements

- Python 3.x
- No external libraries required — uses only Python's standard library (`datetime`, `time`, `math`, `random`, `uuid`, `string`)

## ▶️ How to Run

```bash
python PRO_7.py
```

Then follow the on-screen menu prompts to choose an option.
